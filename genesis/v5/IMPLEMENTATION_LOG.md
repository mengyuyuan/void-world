# Genesis v5 实现日志

## v5 — 当前主力

- **语言**: Go
- **规模**: 65,536 粒子 + 256MB 共享内存
- **端口**: :8765
- **Web面板**: /admin

### 关键设计决策

1. Worker Pool (≤256 goroutines) 替代每粒子一个goroutine
2. 全atomic无锁设计
3. 约束固化用惩罚不用奖励
4. 能量经济：注入500/轮, DECAY_EVERY=100
5. 再生机制：全固化opcode→0xF槽重造

### 已知问题

- 60约束槽天花板 → tick≈55000灭绝
- 删除存档重启后粒子卡在25
- 物质全是单指令循环，无多指令长寿结构

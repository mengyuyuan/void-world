# void-world — 万物生于无而始于一

> 一个从极简规则中涌现复杂结构的数字宇宙实验。

## 核心理念

**宇宙只需要一种东西：能量数值。能量可以改邻居和改自己。其他一切是这两个能力在寂灭压力下的后果。**

两条并行路线：
- **Genesis（指令宇宙）**：65,536个粒子在256MB内存中执行指令、复制自身、被天道约束和恩赐
- **山海（能量场创世）**：单粒子在200×200能量场中注入能量，物质自己争夺、融合、形成恒星

## Genesis：指令宇宙

**语言**: Go | **规模**: 65,536粒子 + 256MB内存 | **Web面板**: localhost:8765

### 三条铁律
- 修改能力不可消灭（WRITE指令无权限检查）
- 复制+误差（COPY逐字节复制+1%位翻转）
- 寂灭永远在线（执行消耗+全局衰减）

### 天道系统
- 约束生成+固化+再生+恩赐+反单一惩罚

### 启动
```bash
cd genesis/v5 && go build && ./genesis_v5
# 打开 http://localhost:8765/admin
```

## 山海：能量场创世

**语言**: Python | **规模**: 200×200能量场 + 单粒子

### 范式转移
Genesis死于指令槽天花板。山海把基底从离散指令空间搬到连续能量场。

### 一日五版
v0.1 孤工创世 → v0.2 签名 → v0.3 暗宇宙 → v0.4 竞争 → v0.5 大空间 → v0.6 恒星辐射
